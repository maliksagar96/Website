We start with the simplest of the differential equation, which is a 1D diffusion equation. And we discretize them in the simplest way possible. We are using a forward difference in time and backward difference in space. Both of them are first order approximations.

It is called a 1D diffusion equation and I call it a 1D wave equation. This step is obtained in between deriving a wave equation. Where that equation becomes an equation symmetrical in space and time. This equation is not yet symmetrical in space and time.

The initial condition that we gave is a square pulse. Theoretically this pulse should travel from left to right without any distortion in the shape of the equation. Whereas when we solve this equation this is getting distorted. Which is not the true behaviour of it.
